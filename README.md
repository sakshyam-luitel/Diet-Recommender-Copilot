# Diet-Recommender-Copilot

A comprehensive application designed to recommend diet plans based on user preferences and health goals.

## Project Structure

```plaintext
Diet-Recommender-Copilot/
├── docs/
│   ├── index.md
│   └── user_guide.md
├── src/
│   ├── data/
│   │   └── database.py
│   ├── models/
│   │   └── diet_model.py
│   ├── services/
│   │   └── recommendation_service.py
│   ├── tests/
│   │   └── test_recommendation_service.py
│   ├── main.py
│   └── utils.py
├── requirements.txt
├── README.md
└── .gitignore
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/sakshyam-luitel/Diet-Recommender-Copilot.git
   cd Diet-Recommender-Copilot
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python src/main.py
   ```
